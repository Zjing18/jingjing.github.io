Major core courses at university

Principles of Computer Networks

"Principles of Computer Networks" covers the fundamentals of network architecture, protocols, and operations, including TCP/IP, routing, Ethernet, and network security. Students gain both theoretical knowledge and practical skills in designing and maintaining networks.

Algorithms and Data Structures

Algorithms and Data Structures introduces fundamental techniques for solving computational problems efficiently. It covers essential data structures (arrays, trees, graphs, hash tables) and algorithms (sorting, searching, recursion). The course emphasizes complexity analysis and practical implementation to develop scalable and optimized solutions.

Big Data Platform and Programming

Big Data Platform and Programming introduces technologies and methods for processing large-scale data. It covers platforms like Hadoop and Spark, programming models such as MapReduce, and tools for distributed storage and parallel computing. The course focuses on practical skills to build and optimize big data solutions.

Computer Organization Principles

Computer Organization Principles explores the fundamental structure and functionality of computer systems. It covers CPU architecture, memory hierarchy, instruction execution, and I/O systems. The course connects hardware design to software performance and efficiency.

Data Analysis and Visualization

This course explores the tools and frameworks widely used in both industry and academic for data collection and analysis,with a focus on the Python programming language.
